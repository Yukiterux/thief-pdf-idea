## 使用教程

1).到release中下载jar包;

2).打开IDEA,找到setting 中的plugin，点击Install Plugin from disk选择下载的jar包，安装并重启IDEA;

3).进入IDEA的Setting页面，找到Other Settings\Thief-Book Config选项，选择PDF文件;

4).愉快的摸鱼吧！！！！

5).当你不小心将窗口关闭时，可以在WIndow菜单下选择show thief重新打开;
